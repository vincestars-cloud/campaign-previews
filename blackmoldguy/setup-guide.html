CRM Setup Guide

The Black Mold Guy — One-time Google Sheets setup (15 minutes)

What this does

Your CRM stores lead data in your own Google Sheet — not anyone else's system. Once set up, all leads from your website go directly into your Sheet. The CRM dashboard reads from that Sheet.

This means: your data is yours. You can see leads in Google Sheets any time, share it with a VA, export it to Excel, or keep it forever — independent of anyone else.

1
Create Your Google Sheet
The spreadsheet that will store all your leads
1
Go to Google Sheets
Open sheets.google.com and log in with your Google account (e.g. theblackmoldguy@gmail.com)
2
Create a new blank spreadsheet
Click + Blank → rename it "Black Mold Guy CRM" at the top
3
Rename the first sheet tab to "Leads"
Right-click on Sheet1 at the bottom → Rename → type Leads → press Enter
The Apps Script will automatically add headers and formatting the first time it runs. You don't need to add any columns manually.
✅
Sheet created. Leave this tab open.
2
Add the Apps Script Backend
This turns your Sheet into a live database your CRM can read and write
1
Open Apps Script from your Sheet
In your Google Sheet → top menu → Extensions → Apps Script. This opens a new tab.
2
Delete the existing code and paste the backend script
Select all code in the editor (Ctrl+A) → delete it → paste the full script below. Then click the floppy disk icon (Save) or press Ctrl+S.
Apps Script Code — copy and paste into the editor:
// THE BLACK MOLD GUY — CRM Backend // Paste this in Extensions → Apps Script, then Deploy as Web App const SHEET_NAME = 'Leads'; const COL = { ID:1, CREATED:2, NAME:3, EMAIL:4, PHONE:5, STATUS:6, SERVICE:7, SOURCE:8, NOTES:9, SEQ_NAME:10, SEQ_START:11, NEXT_EMAIL:12, EMAIL_STEP:13 }; function getSheet() { const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) { sheet = ss.insertSheet(SHEET_NAME); sheet.getRange(1,1,1,13).setValues([['ID','Created','Name','Email','Phone','Status','Service','Source','Notes','Seq Name','Seq Start','Next Email','Email Step']]); sheet.setFrozenRows(1); sheet.getRange(1,1,1,13).setBackground('#1a3a2a').setFontColor('white').setFontWeight('bold'); sheet.setColumnWidth(1,180); sheet.setColumnWidth(2,160); sheet.setColumnWidth(3,140); } return sheet; } function doGet(e) { const action = e.parameter.action || 'getLeads'; try { if (action === 'getLeads') return respond(getLeads()); if (action === 'addLead') return respond(addLead(e.parameter)); if (action === 'updateStatus') return respond(updateStatus(e.parameter)); return respond({error: 'Unknown action'}); } catch(err) { return respond({error: err.toString()}); } } function doPost(e) { try { const data = JSON.parse(e.postData.contents); if (data.action === 'addLead') return respond(addLead(data)); if (data.action === 'updateStatus') return respond(updateStatus(data)); if (data.action === 'saveNotes') return respond(saveNotes(data)); return respond({error: 'Unknown action'}); } catch(err) { return respond({error: err.toString()}); } } function respond(data) { return ContentService.createTextOutput(JSON.stringify(data)).setMimeType(ContentService.MimeType.JSON); } function getLeads() { const sheet = getSheet(); const lastRow = sheet.getLastRow(); if (lastRow <= 1) return {results: []}; const rows = sheet.getRange(2,1,lastRow-1,13).getValues(); const leads = rows.filter(r=>r[0]).map(r=>({ id:r[0], created:r[1], name:r[2], email:r[3], phone:r[4], status:r[5]||'New Lead', service:r[6], source:r[7], notes:r[8], seq_name:r[9], seq_start:r[10], next_email:r[11], email_step:r[12] })); return {results: leads}; } function addLead(data) { const sheet = getSheet(); const id = Utilities.getUuid(); const now = new Date().toISOString(); sheet.appendRow([id,now,data.name||'',data.email||'',data.phone||'','New Lead',data.service||'',data.source||'Website',data.notes||data.message||'','','','',0]); return {success:true, id, created:now}; } function updateStatus(data) { const sheet = getSheet(); const lastRow = sheet.getLastRow(); if (lastRow<=1) return {error:'No leads'}; const ids = sheet.getRange(2,1,lastRow-1,1).getValues(); const idx = ids.findIndex(r=>r[0]===data.id); if (idx===-1) return {error:'Not found'}; const row = idx+2; sheet.getRange(row,6).setValue(data.status); if (data.notes!==undefined) sheet.getRange(row,9).setValue(data.notes); if (data.status==='No Sale - Follow Up') { const today=new Date(), next=new Date(today); next.setDate(today.getDate()+2); sheet.getRange(row,10).setValue('No Sale Follow Up'); sheet.getRange(row,11).setValue(today.toISOString().split('T')[0]); sheet.getRange(row,12).setValue(next.toISOString().split('T')[0]); sheet.getRange(row,13).setValue(1); } return {success:true}; } function saveNotes(data) { const sheet = getSheet(); const lastRow = sheet.getLastRow(); if (lastRow<=1) return {error:'No leads'}; const ids = sheet.getRange(2,1,lastRow-1,1).getValues(); const idx = ids.findIndex(r=>r[0]===data.id); if (idx===-1) return {error:'Not found'}; sheet.getRange(idx+2,9).setValue(data.notes); return {success:true}; }
3
Set up a daily email trigger (optional — for automated follow-up emails)
In Apps Script → left sidebar → Triggers (clock icon) → + Add Trigger → choose function runEmailScheduler → Time-driven → Day timer → 9am. Click Save.
This runs every morning and automatically sends follow-up emails to "No Sale" leads on the right schedule. Skip this step if you don't want automated emails yet.
✅
Script saved. Now deploy it.
3
Deploy as Web App
This gives you the URL that connects your CRM to the Sheet
1
Click Deploy → New Deployment
In the Apps Script editor → top right → Deploy button → New Deployment
2
Configure the deployment settings
Click the gear icon next to "Select type" → choose Web App

Execute as: Me (your Google account)
Who has access: Anyone

Click Deploy
When prompted, click "Authorize access" and follow the prompts. Google will ask you to review the permissions — click "Allow." This is required for the script to read/write your Sheet.
3
Copy the Web App URL
After deploying, you'll see a URL that starts with https://script.google.com/macros/s/... — copy it. This is your CRM's backend URL.
✅
You have your Web App URL. One more step.
4
Connect the URL to Your CRM
This activates the live connection between your dashboard and your Sheet
1
Send your Web App URL to your setup team
Email or message the URL to vince@inleap.ai with the subject "BMG Apps Script URL". They'll update your CRM dashboard in 15 minutes and send you a confirmation.
Once the URL is in place, your CRM dashboard will load leads from your Sheet automatically. New form submissions will also go directly into your Sheet.
2
Test it — open your CRM and check for your Sheet
Go to your CRM dashboard → vincestars-cloud.github.io/campaign-previews/blackmoldguy/crm.html → log in → if you see "No leads yet" (not an error), the connection is working.
✅
Setup complete. Your leads now live in your Google Sheet forever.
