Start here. This guide walks you through exactly where to click in ManyChat to get your AI bot live — from zero. Do these in order. It takes about 30 minutes the first time.
1
Enable ManyChat AI (AI Assistant)
Turn on the AI and connect it to your Instagram/Facebook
1
Go to your ManyChat dashboard
Log in at app.manychat.com → select your page (The Black Mold Guy)
2
Open AI Settings
Left sidebar → Settings → scroll to AI Assistant section
If you don't see "AI Assistant" in Settings, you may need to enable it: go to Settings → Features and toggle on "AI Assistant"
3
Toggle AI Assistant ON
Click the toggle next to "Enable AI Assistant" to turn it green
4
Paste the System Prompt
You'll see a large text field labeled "System Prompt" or "AI Instructions." Click tab 1. System Prompt above, copy the full prompt, and paste it here.
5
Save
Click Save at the bottom of the page
✅
Your AI is now live. Anyone who DMs your Instagram or Facebook page will get AI-powered responses using your system prompt.
2
Create Keyword Flows (the 5 scripts)
Each flow fires when someone DMs a specific word — like "price" or "free inspection"
1
Open the Flows section
Left sidebar → Automation → Flows
2
Create a new flow
Click + New Flow (top right) → name it "Flow 1 — Free Inspection"
3
Set the trigger (keyword)
In the flow builder, click on the starting block → choose Keyword as trigger → type each keyword from the flow (e.g. FREE INSPECTION, FREE CHECK). Add each one with Enter.
Set it to match "Contains" not "Is" — this way it fires even if someone types "Can I get a free inspection?"
4
Add the bot messages
Click the + Add Step button → choose Send Message → paste the message from tab 2. Conversation Flows. Add Quick Reply buttons for each option listed.
5
Publish and repeat for all 5 flows
Click Publish in the top right. Then create flows 2, 3, 4, and 5 the same way using the scripts in tab 2.
✅
5 flows created. Now your bot responds instantly when people ask about prices, health symptoms, free inspections, or submit your form.
3
Set Up Saved Replies (your quick-response library)
These are for YOU to use — one tap sends a full script from your phone
1
Open Saved Replies
Left sidebar → Live Chat → Saved Replies
On mobile: open the ManyChat app → any conversation → tap the lightning bolt icon ⚡ to access saved replies while chatting
2
Add each reply from tab 3
Click + Add Saved Reply → give it a short name (e.g. "Can't Afford", "Bleach", "Scheduling") → paste the script
3
How to use while chatting
In any DM conversation, type / in the reply box → your saved replies appear → select the right one → send. Or tap the ⚡ icon on mobile.
✅
10 saved replies loaded. Now when someone DMs you personally, you can respond with a perfect script in one tap.
4
Set Up Comment Automation (auto-DM from comments)
When someone comments specific words on your posts, they automatically get a DM
1
Open Growth Tools
Left sidebar → Growth Tools → Comment Growth Tool (sometimes listed as "Comments")
2
Create a new Comment Automation
Click + New Comment Growth Tool → choose Instagram or Facebook
3
Set the trigger keywords
Choose "When someone comments containing..." → add: FREE, INSPECT, HELP, DM ME, INFO
Also set up a public reply to the comment: "Sent you a DM with all the info! 🌿" — this keeps the conversation visible to others and shows you're responsive.
4
Connect to your Flow 1
Under "Send DM" → select Connect to Flow → choose "Flow 1 — Free Inspection". This sends the full booking script automatically.
5
Apply to posts
Choose: All Posts (recommended) or specific posts. Click Publish.
✅
Now every time someone comments "info" or "DM me" on your Reels, they automatically get your booking flow in their DMs — while you sleep.
5
Set Up the 24-Hour Follow-Up Sequence
Automatically re-engages people who DM'd but didn't book
1
Open Flow 4 (24-Hour Re-Engagement)
Go to Automation → Flows → open "Flow 4 — 24-Hour Re-Engagement"
2
Set the trigger to "First interaction"
Set trigger to First Interaction with Page → then add a Delay step of 24 hours → then add the re-engagement message from tab 2.
3
Add a condition (don't message if already booked)
Add an Condition before the message: check if the user has a custom field "booked = yes" — if yes, skip the message. (Set this field via a button in your booking confirmation flow.)
✅
24-hour follow-up active. People who ghost you get a gentle nudge automatically the next day.
